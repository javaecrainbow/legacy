誠品球衣有限公司　

  • 誠品成立於2014年2月，從事各運動球衣的製作、團隊制服印製、來貨加工等服務，是首家提供三小時印製T-shirt及球衣服務的香港公司。
  • 本公司所提供的服務包括足球、籃球、手球、排球球衣、T-shirt、風褸、外套、足球/籃球裝備、來貨加工、繡花章等，T-shirt方面， 我們已與國外的優質品牌建立良好合作關係，如美國品牌Gildan和Fruit of the Loom等，以引入更多高質素的t-shirt，給予客人更多的選擇。 球衣方面，除了ADIDAS和NIKE等品牌，本公司與歐洲方面的Adidas和Nike代理達成戰略協議，於2015年開始，全面由引進歐洲直接入口的adidas，nike和puma等品牌的球衣，增加產品的多樣性，而且有本公司將會設立自家品牌，可讓客戶自行配搭球衣款式和顏色，提高訂造的彈性。 貨品供應方面，為使貨源充足，本公司的貨倉，約有50000套或以上的存貨；另外，在製作方面，本公司擁有良好的印製設備，包括絲印和熨畫，絲印方面，由超過十年的印花技師的團隊組成，漿料多樣化，可做不同種類的印花，包括絨面絲印，熨金，反光和螢光色等，熨畫具有四部最新型號的熨畫機和兩部價值十多萬的熨畫機器，做到一條龍服務，並提升印製速度，從而提高生產力，以做到「加工不外求」的成效。(本公司一天的印製速度可達至2000件T-shirt或以上)
  • 若客戶需要送貨服務，以單一發票計算，價格只需超過港幣3000元；或已付全數(現金/銀行轉賑/支票均可)，本公司即可享有免費送貨服務， 送貨範圍包括香港、九龍、新界及離島地區。 國外客戶，可透過電郵方式落單，銀行轉賑/支票/Paypal付款，貨品會經值得信賴的速遞公司順豐速運、香港郵政、DHL或FEDEX等運送。(國外客戶須自付郵費。)
  • 本公司曾贊助南華排球隊，青協及其他業餘球隊，希望對社會作出回饋，為社會出一份力。如有任何球隊或機構尋求贊助，可將計劃書電郵至sales@legacy-sport.com。
